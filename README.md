# RARS_IWH2

# Программа вычисления корня пятой степени на RISC-V ассемблере

## Описание проекта

Данный проект представляет собой программу на ассемблере RISC-V для вычисления корня пятой степени с использованием быстро сходящегося итерационного алгоритма. Программа обеспечивает точность вычислений не хуже 0.1% и включает комплексную систему автоматизированного тестирования.

## Основные возможности

- **Вычисление корня 5-й степени** по итерационному алгоритму Ньютона
- **Автоматизированное тестирование** с покрытием различных граничных случаев
- **Обработка ошибок** для отрицательных входных данных
- **Интерактивный режим** для ручного ввода чисел
- **Использование макросов** для ввода/вывода и управления программой
